# FalilNifal GitHub Profile Folder

Upload this full folder content to your GitHub profile repository named exactly:

```text
FalilNifal
```

## Correct folder structure

```text
FalilNifal/
  .github/
    workflows/
      generate-pacman.yml
      generate-snake.yml
  cosmic-skill-map.png
  README.md
```

## After uploading

1. Go to your GitHub profile repository.
2. Go to `Settings → Actions → General`.
3. Under `Workflow permissions`, select `Read and write permissions`.
4. Click `Save`.
5. Go to `Actions`.
6. Open `Generate Pacman`.
7. Click `Run workflow`.
8. Wait until it finishes successfully.
9. Refresh your GitHub profile.

The Pac-Man graph will show only after the workflow creates the `output` branch.
